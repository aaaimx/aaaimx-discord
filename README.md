# AAAIMX AAAIMX Discord Bot
Discord Bot for AAAIMX server

### Prerequisites 📋

- Node.js
- Chrome
- VSCode

### Installing

```
npm install
```


## Deployment 📦

```
nodemon --inspect index.js 
```

## Built With

* [Discord.js](https://discord.js.org/#/) - A powerful library for interacting with the Discord API
